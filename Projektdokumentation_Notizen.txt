1. Einleitung
Die folgende Projektarbeit befasst sich mit der Anwendung künstlicher Intelligenz für die Softwareentwicklung am Beispiel von ChatGPT und Github Copilot. 
Dabei wird insbesondere ein Fokus darauf gelegt, wie man im Studium den Studierenden dieses Thema näher bringen und damit einen Mehrwert in der Ausbildung schaffen kann. 

2. Aufgabenstellung und Ziele
Aufgabenstellung:
- Sammlung von Erfahrung im Umgang mit KI in der Softwareentwicklung und Einblick in verschiedene Bereiche 

Ziele:
- Testen von verschiedenen Produkten auf dem Markt und Empfehlung für den Einsatz im Studium
- Erstellung von praktischen Anwendungsbeispielen anhand von Praktikumsaufgaben für das Modul C++ Programmierung

3. Themengebiete
3.1 ChatGPT von OpenAI
3.2 Github Copilot
3.3 Vergleich ChatGPT und Github Copilot
3.4 Code Refactoring mit KI
Dieser Abschnitt behandelt das Code Refactoring mit künstlicher Intelligenz. Es wird anhand von Beispielen ein schrittweises Vorgehen dargestellt um die Struktur und Verständlichkeit eines gegebene Codes zu verbessern.
